# Gym Bro Recipes - Informational Resource

An informational blog-style website providing educational content about bodybuilding training phases: Maintaining, Cutting, and Bulking.

## Overview

This site serves as a point of information based on personal experience and research. It is designed as a clean, informational resource without social media integration or call-to-action elements.

## Features

- **Clean, Centralized Layout**: Modern design with clear white space and organized content
- **Bold Typography**: Clear, readable fonts with short paragraphs for easy scanning
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices
- **Smooth Navigation**: Fixed navigation bar with smooth scrolling between sections
- **Informational Focus**: No CTAs or social media - purely educational content
- **Disclaimer Sections**: Clear disclaimers about the informational nature of the content

## Structure

- `index.html` - Main HTML structure with all content sections
- `styles.css` - Complete styling with responsive design
- `script.js` - Navigation and interactivity functionality

## Sections

1. **Maintaining** - Information about maintaining current physique
2. **Cutting** - Guidance on reducing body fat while preserving muscle
3. **Bulking** - Strategies for muscle growth and strength gains

## Disclaimer

This site is for informational purposes only. The content is based on personal experience and should not be used as the sole method for achieving fitness goals. Always consult with healthcare professionals or certified trainers before beginning any new fitness or nutrition program.
